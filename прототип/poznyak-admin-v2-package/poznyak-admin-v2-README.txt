PZK Admin V2
Дата: 17.07.2026

Основные файлы:
- poznyak-admin-prototype-v2.html - интерактивный автономный прототип.
- poznyak-admin-prototype-v2-codex-task.md - обновленное техническое задание для Codex.

Как открыть прототип:
1. Скачайте HTML.
2. Откройте файл в Chrome или Edge.
3. Используйте левое меню для перехода между разделами.

Что интерактивно:
- переходы между разделами;
- канбан и таблица проектов;
- открытие карточки проекта;
- вкладки карточки;
- задачи и фильтры;
- чек-лист завершения;
- подтверждение второй оплаты;
- разблокировка и отправка спецификаций;
- редактирование услуги в drawer;
- конструктор опросника;
- drawer настроек вопроса на ширине 1024 px;
- уведомления и мобильный sidebar.

Production не изменялся.
