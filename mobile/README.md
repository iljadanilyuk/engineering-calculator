# Mobile

Mobile is deferred for the Poznyak Engineering Calculator.

The v1 scope is web-only:

- `website` for the public landing page and calculator.
- `webapp` for the authenticated admin cabinet.
- `backend` and `packages/contracts` for API, persistence, auth, proposals, notifications, and shared validation.

Do not add mobile runtime work unless `task.md` is updated with an explicit mobile requirement.
